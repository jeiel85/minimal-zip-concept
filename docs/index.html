Rust 기반 무손실 압축의 모든 원리를 담다

MZC (Minimal Zip Concept)는 단순한 압축기를 넘어, 바이트 단위 및 비트 단위의 직렬화, RLE, 사전 대치, LZ77 슬라이딩 윈도우, 점프 상대 주소 변환(BCJ), 델타 필터 및 Canonical Dynamic Huffman Coding까지 무손실 압축의 실전 구조를 설계하며 학습하는 러스트 지향 압축 엔진입니다.

주요 고도화 기능 명세

압축 오버헤드를 극적으로 배제하고 성능을 조율하기 위해 추가된 주요 요소들입니다.

🪙

비트 레벨 스트림 패킹

블록마다 발생하는 바이트 단위(1B) 타입 오버헤드를 극복하기 위해, 압축 블록 8개마다 2바이트 플래그(2비트 x 8)로 비트 스트림화하여 선두에 기록함으로써 타입 공간을 약 15~25% 절약합니다.

⚡

지연 매칭 및 압축 레벨

그리디 방식의 즉시 매칭을 개선하여, 다음 바이트에서 더 효율적인 LZ77 일치가 발견될 경우 결정을 이월하는 지연 매칭(Lazy Matching) 기법을 탑재하고 압축 레벨(1~9)에 맞춰 연산 강도를 조절합니다.

🎹

데이터 원시 전처리 필터

바이너리의 엔트로피를 물리적으로 떨어뜨리는 x86 기계어 타겟 BCJ(Branch-Call-Jump) 상대 번지 대치 필터와 WAV/BMP 등 연속 파형 바이트의 차분값을 계산하는 Delta Filter를 제공합니다.

MZC 하위 호환성 포맷 아키텍처

MZC 엔진은 버전 1부터 최신 버전 5까지 유연한 이중 파싱 분기(Double-Dispatch)를 수행합니다.

MZC1 Specification

RLE 기반 초급 포맷

압축 알고리즘의 기초인 RLE(Run-Length Encoding)를 구현하여, 바이트 단위 직렬화 파이프라인의 구조를 세운 기초 명세입니다.

  • 54바이트의 고정 파일 헤더 제공
  • 런 블록(Type 0x01)과 리터럴 블록(Type 0x00)만 사용
  • 단일 스레드 기반의 동작 및 기본 SHA-256 체크섬

MZC2 Specification

하이브리드 병렬 사전식 압축

자주 등장하는 바이트 시퀀스를 사전에 등록하고, 토큰 번호로 일치 항목을 매핑하는 멀티스레드 병렬 압축 규격입니다.

  • Rayon 라이브러리를 동원한 1MB 청크 단위 병렬 맵-리듀스
  • 사전 인프라 추가로 파일 헤더 크기가 56바이트로 확장
  • RLE와 사전 토큰(Token Block, Type 0x02)의 Hybrid 탑재

MZC3 Specification

LZ77 슬라이딩 윈도우 지원

데이터의 이전 윈도우 공간에서 반복 영역을 찾아 오프셋 거리(Distance)와 길이(Length) 정보로 인코딩하는 LZ77 사양이 통합되었습니다.

  • 백레퍼런스 블록(BackRef Block, Type 0x03) 추가
  • 32KB 크기의 슬라이딩 윈도우 룩어헤드 버퍼 탐색
  • 정적 허프만 코딩을 연동한 2차 비트 스트림 추가 압축

MZC4 Specification

Canonical Dynamic Huffman Coding

사전 정의된 고정 허프만 트리를 폐기하고, 청크 파일의 엔트로피 빈도수를 실시간 계산하여 최적화된 Canonical 트리를 동적으로 생성합니다.

  • Dynamic Huffman Tree 헤더 크기를 20~40B 미만으로 억제하는 독자 트리 RLE 탑재
  • 심볼 트리 깊이를 30비트 이하로 보장하는 정합 설계
  • 소용량 파일에서 압축 효율 극대화

MZC5 Specification

비트 레벨 스트림 패킹 및 하이엔드 전처리

바이트 정렬 한계를 완전히 무너뜨리고 8개 단위 플래그로 묶은 비트스트림 구조로 압축 메타데이터 오버헤드를 원천적으로 제거한 최상위 버전입니다.

  • 8블록 단위 16비트 플래그 스트림으로 타입 접두사 완전 대체
  • BCJ 상대 주소 대치 필터 및 Delta 필터 탑재
  • 디스크 쓰기/읽기 및 디코딩 연산 전체에 멀티스레드 정밀 바운드 예외 차단 기술 적용

실행 가이드 및 사용법

CLI 커맨드와 미려한 GUI 환경을 모두 동일한 Rust 코어 라이브러리로 이용할 수 있습니다.

PowerShell / Terminal Rust Cargo
# 1. 깃 클론 및 폴더 이동 git clone https://github.com/jeiel85/minimal-zip-concept.git cd minimal-zip-concept # 2. 러스트 빌드 (릴리즈 모드) cargo build --release # 3. CLI를 이용한 고성능 MZC5 압축 실행 (레벨9, Delta 활성화, BCJ 활성화) ./target/release/mzc compress input.bin output.mzc5 -m lz77 -e dynamic -l 9 --delta --bcj # 4. 압축 파일 디코딩 및 복원 검사 ./target/release/mzc decompress output.mzc5 restored.bin # 5. 그래픽 GUI 성능 모니터 대시보드 뷰어 기동 ./target/release/mzc gui
